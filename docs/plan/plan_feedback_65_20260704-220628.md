# 修复计划卡片更多操作菜单覆盖布局

## 需求概要

计划卡片的更多操作菜单当前作为绝对定位子层从底部向上展开，容易覆盖查看任务、验收 pending 以及相邻卡片内容。本计划优先采用稳妥 UI 修复：让更多操作按钮进入 plan-actions 正常 flex 布局，菜单默认向下展开，并补充静态回归断言，确保菜单语义、停止/删除项与非覆盖定位保持稳定。

  - 验收要点：更多操作按钮与并发执行、阅读全文、查看任务、验收状态等操作共同参与 .plan-actions 的 flex 布局，打开或关闭菜单时不压住同一行文字。

# 反馈 #67：AI 配置默认模型改为 gpt-5.5

## 需求概要

根据反馈 #67，需要将应用内 AI 配置的 OpenAI 默认模型从 gpt-4o 调整为 gpt-5.5。结合代码现状，默认模型分布在后端 AI 配置服务、数据库默认 settings、前端表单默认值、Provider 默认模型工具以及对话状态兜底配置中；实现时应只改变默认值和默认展示，不覆盖用户已经保存的自定义模型。

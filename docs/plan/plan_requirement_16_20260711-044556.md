# P09：数据库所有权原子切换工具与副本演练

## 需求概要

在 P04–P08 全部通过的前提下，依据 D:\docs\autoplan-go-backend-separation-plan.md v1.0 和现有 Node/sql.js 调用现状，实现仅面向脱敏 fixture、临时库或明确副本的数据库所有权原子切换工具链。补齐旧 Loop、Chat、脚本与执行器运行时所需的 Go application/API 命令，以 GoDataClient 取代 Go 模式下的 Node 直连数据库，并由 Electron daemon supervisor 和维护模式编排确保冻结任务、排空进程、备份审计、迁移、唯一锁、读写冒烟和 UI 开放按失败关闭顺序执行。阶段交付包含自动化故障注入、接近真实规模副本兼容验证、回滚手册、真实命令与退出码、脱敏证据、文件清单和风险；绝不自动发现或操作真实 Electron userData，也不允许 Node/sql.js 与 Go 同写同库或旧 sql.js 热打开新 schema。

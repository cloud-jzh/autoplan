# 需求/反馈输入框支持 @ 引用其它需求与反馈

## 需求概要

在工作区的需求和反馈 Composer 中增加 @ 引用能力：用户输入 @ 后可从当前项目的需求、反馈记录中检索并插入规范引用；已提交内容中的引用可识别、展示并定位到对应记录；计划生成时解析正文中的引用并注入被引用记录摘要，使引用不仅是文本标记，也能为后续计划拆解提供上下文。实现应优先复用现有 intake 快照、草稿、搜索定位和计划生成链路，避免新增数据库迁移，引用以可持久化的正文文本形式兼容既有 create/update/MCP 流程。

# 验收视图支持阅读 Plan 全文（复用 PlanReaderModal）

## 需求概要

需求 #95「验收环节，也要看到 plan，阅读全文」。当前验收视图（AcceptanceView）的待验收/已验收计划卡只展示计划文件名、标题、进度与任务列表，无法查看计划 Markdown 正文。项目已有可复用的「阅读全文」机制：页面级 PlanReaderModal（WorkspacePage 中以 planReadState 单实例挂载）+ useWorkspaceController 暴露的 openPlanReader(plan) / openTaskPlanReader(task)，PlanList 与 TaskList 已分别经 onOpenReader / onOpenPlan 接入。本次只需在验收视图的计划卡（及其下任务行）增加「查看 Plan 全文」触发器，并在 WorkspacePage 把 openPlanReader / openTaskPlanReader 透传给 AcceptanceView，点击即复用既有只读 Modal 展示该计划全文，不新增第二个 reader 实例、不改后端验收逻辑。

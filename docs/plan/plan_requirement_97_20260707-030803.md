# 计划中断后可切换CLI工具执行 & 计划结束后支持重新创建/执行/附加内容

## 需求概要

为 AutoPlan 的计划生命周期增加两个核心能力：1) 计划执行中断或停止后，允许用户在恢复前切换 CLI 执行工具（codex/claude/opencode/oh-my-pi），使同一计划可在不同后端继续执行；2) 计划完成（completed）后，支持重新创建（基于原需求重新生成计划）、重新执行（重置任务状态重新运行）以及附加内容（向已完成计划追加新任务并重新激活），避免需求变更后必须手动创建新计划。

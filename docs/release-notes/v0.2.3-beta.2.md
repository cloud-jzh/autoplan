# AutoPlan v0.2.3-beta.2

> **Beta 未测试版本**：这是面向快速分发的预发布版本，未经过完整人工测试，不等同于稳定正式版，也不承诺零缺陷。

## 近期更新摘要

- 基于 `v0.2.3-beta.1` 重新发布，包含全局 AI 配置、多会话对话、搜索弹层、侧栏交互、设置页与近期工作区 UI 更新。
- 修复 GitHub Actions Release workflow 的 Linux 安装包匹配规则：兼容 electron-builder 实际输出的 `linux-x86_64.AppImage` 与 `linux-amd64.deb` 文件名。
- 保留远端已合并的 Linux 构建类型修复，继续包含 `.nvmrc`、lockfile 与 `aiConfigUpdate` 类型适配。

## Beta 风险提示

本 beta 版本用于更快分发近期修复和功能更新。当前策略是：核心检查、构建和 smoke 通过后即可发布，以减少等待时间。因此，下载测试或日常使用中仍可能遇到功能缺陷、平台差异或安装包问题。
